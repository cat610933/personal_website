# Assets

Use this folder for files that tutorial pages or project pages include.

- `code/`: runnable scripts, notebooks, and small reproducible examples
- `images/`: figures, diagrams, screenshots, and exported plots
- `formulas/`: equation images, derivation notes, or formula-related assets

Homepage images:

- `images/profile/profile.jpg`: profile photo shown in the hero area
- `images/icon/icon.png`: small decorative/personal icon shown beside the profile photo

Prefer small files for GitHub Pages. Large videos or datasets should live elsewhere and be linked from the page.
