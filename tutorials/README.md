# Tutorials

Use `tutorials/_template.html` as the starter for new tutorial pages.

Suggested workflow:

1. Copy `tutorials/_template.html`.
2. Rename it, for example `tutorials/my-new-topic.html`.
3. Edit the title, sections, PDF path, and download links.
4. Add a new entry to the `tutorials` array in `scripts/site-data.js`.

Assets usually go here:

- PDF notes: `assets/pdfs/`
- Code: `assets/code/`
- Images or plots: `assets/images/`
- Download bundles: `assets/downloads/`
